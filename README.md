# Squid233's Logger

This is squid233's logger.

## Configuration

To use this program, you need a config file. The file name is `squidlog.cfg`.

You can see [squidlog.cfg](squidlog.cfg).

squidlog.cfg syntax:

    logger-level=the logger level sum

|logger-level|max can be display|
|---|---|
|-1|unable|
|0|message(no prefix)|
|1|info|
|2|warn|
|4|error|
|8|fatal|
|16|debug|
